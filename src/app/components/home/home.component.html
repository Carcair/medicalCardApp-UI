This is a Medical Card Application

Purpose of this application is for medical employee and/or institutions is to add info for their visiting patients, and to input their every diagnosis. User can review, add, edit or delete patients from DB, but can only review and input diagnosis. By deleting a patient, you also delete all his diagnosis.
